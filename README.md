# Team Task Manager Frontend

## Overview
The Team Task Manager is a web application designed to help teams manage their tasks efficiently. It provides a user-friendly interface for creating, updating, and tracking tasks, ensuring that team members stay organized and productive.

## Features
- Create, update, and delete tasks
- View tasks in a list format
- Filter tasks by status
- Responsive design for mobile and desktop

## Technologies Used
- **Frontend**: React, TypeScript, Axios, Tailwind CSS
- **Backend**: Node.js, Express, MongoDB, Mongoose

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- MongoDB (local or cloud instance)

### Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   cd team-task-manager
   ```

2. Navigate to the frontend directory:
   ```
   cd frontend
   ```

3. Install the frontend dependencies:
   ```
   npm install
   ```

4. Navigate to the backend directory:
   ```
   cd ../backend
   ```

5. Install the backend dependencies:
   ```
   npm install
   ```

### Running the Application

1. Start the backend server:
   ```
   npm start
   ```

2. In a new terminal, navigate to the frontend directory and start the frontend application:
   ```
   cd frontend
   npm start
   ```

3. Open your browser and go to `http://localhost:3000` to access the application.
